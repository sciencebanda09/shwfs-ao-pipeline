# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.1.x   | ✅ |
| 1.0.x   | ❌ |

## Reporting a Vulnerability

This is a research/simulation codebase. If you find a security issue:

1. Do **not** open a public issue
2. Email the maintainer directly via GitHub profile
3. Include a description and reproduction steps

Response expected within 7 days.
